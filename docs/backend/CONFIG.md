# Backend Configuration

| Variable | Description | Default |
|----------|-------------|---------|
| `CATALOG_DB_PATH` | SQLite catalog generated by `crawler/scripts/import_catalog_db.py`. | `../crawler/data/db/catalog.db` (resolved from backend directory) |
| `USER_DB_ROOT` | Directory with planner snapshots (`course_<id>.json`). | `../crawler/data/user_db` |

To configure:

```powershell
cd backend
copy env.example .env
# edit .env to match your environment if you move the crawler outputs
```

The backend reads these values via `app/config/settings.py`. Update them whenever the crawler data is stored elsewhere (e.g., when deploying to a server with a different directory layout).
